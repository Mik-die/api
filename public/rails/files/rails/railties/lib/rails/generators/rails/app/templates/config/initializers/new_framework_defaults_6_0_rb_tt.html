This file contains migration options to ease your Rails 6.0 upgrade.

Once upgraded flip defaults one by one to migrate to the new default.

Read the Guide for Upgrading Ruby on Rails for more info on each option. Don't force requests from old versions of IE to be UTF-8 encoded. Rails.application.config.action_view.default_enforce_utf8 = false Embed purpose and expiry metadata inside signed and encrypted cookies for increased security.

This option is not backwards compatible with earlier Rails versions. It's best enabled when your entire app is migrated and stable on 6.0.
