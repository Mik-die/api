Module OpenSSL::Netscape

  • ruby/ext/openssl/ossl.c

OpenSSL::Netscape is a namespace for SPKI (Simple Public Key Infrastructure) which implements Signed Public Key and Challenge. See RFC 2692 and RFC 2693 for details.
