Class WEBrick::HTTPServlet::ProcHandler < WEBrick::HTTPServlet::AbstractServlet

  • ruby/lib/webrick/httpservlet/prochandler.rb

Mounts a proc at a path that accepts a request and response.

Instead of mounting this servlet with WEBrick::HTTPServer#mount use WEBrick::HTTPServer#mount_proc:

server.mount_proc '/' do |req, res|
  res.body = 'it worked!'
  res.status = 200
end
