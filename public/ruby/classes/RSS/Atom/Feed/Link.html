Class RSS::Atom::Feed::Link < RSS::Element

  • ruby/lib/rss/atom.rb

Defines a reference to a Web resource. It has the following attributes:

  • href

  • rel

  • type

  • hreflang

  • title

  • length

Reference: validator.w3.org/feed/docs/rfc4287.html#element.link
