Class ActionDispatch::Session::UpgradeSignatureToEncryptionCookieStore < ActionDispatch::Session::EncryptedCookieStore

  • rails/actionpack/lib/action_dispatch/middleware/session/cookie_store.rb

This cookie store helps you upgrading apps that use CookieStore to the new default EncryptedCookieStore To use this CookieStore set

Myapp::Application.config.session_store :upgrade_signature_to_encryption_cookie_store, key: '_myapp_session'

in your config/initializers/session_store.rb

You will also need to add

Myapp::Application.config.secret_key_base = 'some secret'

in your config/initializers/secret_token.rb, but do not remove +Myapp::Application.config.secret_token = 'some secret'+
